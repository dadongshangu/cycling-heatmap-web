这个目录包含了一些示例GPX文件，用于测试和演示骑行热力图生成器的功能。

### 1. sample_beijing_ride.gpx

## 🎯 使用方法

### 快速测试
1. 打开骑行热力图生成器应用
2. 点击"选择文件"或直接拖拽GPX文件到上传区域
3. 选择一个或多个示例文件
4. 点击"生成热力图"按钮
5. 在地图上查看热力图效果

## 🔧 技术细节

### GPX格式规范
- 符合GPX 1.1标准
- 包含完整的元数据信息
- 每个轨迹点包含：
  - 纬度 (lat)
  - 经度 (lon)
  - 海拔 (ele)
  - 时间戳 (time)

### 坐标系统
- **坐标系**: WGS84 (World Geodetic System 1984)
- **格式**: 十进制度数
- **精度**: 小数点后4位 (~11米精度)

### 创建自己的示例文件
如果你想创建自己的示例GPX文件，可以：

1. **使用GPS设备记录**
   - 手机GPS应用

2. **在线工具生成**

3. **手动编写**
   - 参考现有示例文件格式
   - 使用文本编辑器创建
   - 确保XML格式正确

### 示例文件模板
```xml
<?xml version="1.0" encoding="UTF-8"?>
<gpx version="1.1" creator="Your App" xmlns="http://www.topografix.com/GPX/1/1">
  <metadata>
    <name>Your Route Name</name>
    <desc>Route description</desc>
    <time>2024-01-01T00:00:00Z</time>
  </metadata>
  <trk>
    <name>Track Name</name>
    <type>Cycling</type>
    <trkseg>
      <trkpt lat="latitude" lon="longitude">
        <ele>elevation</ele>
        <time>timestamp</time>
      </trkpt>
      <!-- 更多轨迹点... -->
    </trkseg>
  </trk>
</gpx>
```

## 📝 注意事项

1. **文件大小**: 示例文件较小，实际GPX文件可能包含数千个轨迹点
2. **时间间隔**: 示例文件使用1分钟间隔，实际记录通常为1-5秒间隔
3. **精度**: 示例坐标为模拟数据，实际GPS精度可能有所不同
4. **隐私**: 这些是公开的示例路线，不包含个人隐私信息
