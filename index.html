    📖 使用帮助

    🚀 快速开始

    1. 拖拽或选择轨迹记录GPX上传
    2. 调整参数设置（可选）
    3. 点击"生成热力图"按钮
    4. 在地图上查看结果

    ⚙️ 参数说明

    • 地图样式: 选择暗色或浅色背景
    • 线条粗细: 数值越大线条越粗
    • 模糊程度: 数值越大越模糊
    • 透明度: 热力图的透明度
    • 时间范围: 只显示指定时间内的轨迹

    📁 支持的文件格式

    • 轨迹记录GPX (.gpx)
    • 支持批量上传多个文件
    • 支持骑行、徒步、跑步等所有GPS轨迹

    🎨 Strava风格

    热力图使用绿→黄→红的渐变色，频繁经过的路线会显示为红色。

    📖 如何获取轨迹记录GPX？

    🚴 码表设备

    🇨🇳 国产码表品牌

    📱 迈金码表 (MEILAN)

    对应App: 顽鹿竞技

    1. 确保码表数据已通过蓝牙同步到顽鹿竞技App
    2. 打开顽鹿竞技App，进入"活动记录"
    3. 选择要导出的骑行活动
    4. 点击"分享"或"导出"按钮
    5. 选择"GPX格式"进行导出
    6. 保存到手机或直接分享到电脑
    📱 iGPSPORT码表

    对应App: iGPSPORT官方App

    1. 打开iGPSPORT App，同步码表数据
    2. 在"历史记录"中找到要导出的活动
    3. 点击活动详情页面
    4. 选择"导出"→"GPX格式"
    5. 也可以同步到Strava后再导出
    📱 其他国产码表
    • 猫眼(CatEye): 使用CatEye Cycling App导出
    • 速腾(XOSS): 使用XOSS App导出
    • 百锐腾(Bryton): 使用Bryton Active App导出

    🌍 国际品牌码表

    📱 Garmin码表 (Edge系列)
    1. 方法1 - Garmin Connect:
      • 打开Garmin Connect网站或App
      • 找到要导出的活动
      • 点击"导出原始数据"
      • 选择GPX格式下载
    2. 方法2 - USB直连:
      • 用USB线连接码表到电脑
      • 在Garmin/Activities文件夹中找到.fit文件
      • 使用在线工具转换为GPX格式
    📱 Wahoo码表 (ELEMNT系列)
    1. 打开Wahoo ELEMNT App
    2. 在"历史记录"中选择活动
    3. 点击"分享"按钮
    4. 选择"导出GPX"
    5. 也可以同步到Strava后导出

    ⌚ 智能手表

    📱 Apple Watch
    1. 通过第三方App:
      • 使用Strava、WorkOutDoors等App记录
      • 在对应App中导出GPX
    2. 通过健康App:
      • 健康App → 浏览 → 活动
      • 选择"体能训练"
      • 导出健康数据（需要第三方工具转换）
    📱 Garmin手表
    1. 同Garmin码表，使用Garmin Connect导出
    2. 在Garmin Connect中找到手表记录的活动
    3. 导出为GPX格式
    📱 华为/小米手表
    1. 打开对应的运动健康App
    2. 找到运动记录
    3. 查看是否有导出功能
    4. 或同步到第三方平台后导出

    📱 手机应用

    🌍 国际应用

    📱 Strava
    1. 单个活动导出:
      • 打开Strava网站
      • 进入活动详情页
      • 点击右上角"..."菜单
      • 选择"导出GPX"
    2. 批量导出:
      • 进入设置 → 隐私控制
      • 点击"下载或删除你的账户"
      • 请求数据导出（包含所有轨迹记录GPX）
    📱 Komoot / MapMyRide
    1. 登录对应网站
    2. 找到活动记录
    3. 查找导出或下载选项
    4. 选择GPX格式

    🇨🇳 国内应用

    📱 顽鹿竞技
    1. 打开顽鹿竞技App
    2. 进入"活动记录"
    3. 选择要导出的骑行
    4. 点击"分享"或"导出"

    📱 骑记 / 行者 / 咕咚
    1. 打开对应App
    2. 找到运动记录
    3. 查看是否有"导出"或"分享"功能
    4. 部分App可能需要会员才能导出
    5. 也可以尝试同步到Strava后导出
    🏔️ 两步路（户外助手）
    1. 打开两步路App

    3. 选择要导出的轨迹
    4. 点击轨迹详情页右上角"..."菜单
    5. 选择"导出" → "GPX格式"
    6. 也可以通过网页版（www.2bulu.com）登录后导出

    ❓ 常见问题

    Q: 我的设备只能导出.fit文件，怎么办？

    A: 可以使用在线转换工具，如GPSies、GPS Visualizer等，将.fit文件转换为GPX格式。

    Q: 为什么我的App没有导出功能？

    A: 部分App可能需要付费会员才能导出，或者可以先同步到Strava等平台再导出。

    Q: 轨迹记录GPX很大，上传很慢怎么办？

    A: 可以使用GPX编辑工具简化轨迹点，或者分批上传多个小文件。

    Q: 我的迈金码表应该用什么App？

    A: 迈金(MEILAN)码表对应的官方App是"顽鹿竞技"。
